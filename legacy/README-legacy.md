# legacy — animações p5.js

Estes arquivos são a versão anterior deste repositório: animações interativas
em p5.js (peônia 3D e dinheiro/dólar→moeda). Foram preservados aqui quando o
repositório passou a ser o **VWRA Radar** (ver `../README.md`).

- `index.html` — painel/galeria que abre as animações
- `peonia.html` / `peonia-plus.html` — a peônia (original e versão estendida)
- `dolar.html` — nota de dólar interativa
- `dinheiro.html` — versão só visual (nota → moeda de ouro → bitcoin), em loop

Abra qualquer um direto no navegador; rodam standalone (só puxam p5.js da CDN).
O `README.md` original está preservado como `README.md` nesta pasta.
